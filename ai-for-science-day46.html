面向：有编程经验的非 AI 方向工程师

前 45 期拆的是「AI 系统自己怎么工作」。今天换视角：AI 作为科研工具——怎么把「蛋白质长什么样」「哪种新材料稳定」「更快的算法是什么」这些百年难题，变成机器学习能啃的搜索与预测问题。四个概念覆盖生物、物理、化学、数学，共同点一句话：把「昂贵的实验/数值模拟」换成「廉价的神经网络前向」，或把「人类的灵光一现」换成「可验证的自动搜索」。

蛋白质结构预测AlphaFold

结构生物学几何注意力
一句话类比

蛋白质折叠 = 从一维序列反推最终的三维内存布局。像从一段 protobuf schema 定义，反推它在内存里最终会盘绕成什么样的对象图——只不过这里的「布局规则」是物理能量最小化。最妙的一步在进化信息：同一个蛋白在成千上万物种里都有「变体副本」（MSA，多序列比对）。哪两个位置总是一起变异，就说明它们在 3D 里靠得近、互相约束——正是「总是一起 failover 的两个节点，八成在同一可用区」的推理。

它解决什么问题 + 工作机制

痛点：蛋白质功能由 3D 形状决定，但实验测结构（X 射线晶体学、冷冻电镜）一个要花数月到数年、几十万美元。序列好测、形状难测——这个鸿沟卡了生物学 50 年。

AlphaFold2（DeepMind, 2021）的机制分三段：① 把目标序列 + 它的 MSA 进化家族喂进去；② 核心的 Evoformer 用注意力在「序列维度」和「残基对维度」之间反复交换信息——本质是在推断「哪些氨基酸对在空间上耦合」；③ 结构模块用「不变点注意力」直接吐出每个原子的三维坐标，且对整体平移旋转不变（形状不该因坐标系而变）。它在 CASP14 盲测里把预测精度推到接近实验水平，这是几乎没人预料会在那个十年发生的跳跃。

AlphaFold2 数据流（一维序列 → 三维坐标）
目标序列 + MSA 进化家族 → Evoformer (48 层注意力，推断残基对耦合)
↓
结构模块 (不变点注意力，平移旋转不变) → 3D 原子坐标 + pLDDT 置信度

关键副产品：pLDDT——每个残基一个 0-100 的自评置信度，诚实标出「哪段我拿不准」（往往是天然无序区）。有了它，你才敢把预测当假设用。

代码示例
import requests
# AlphaFold DB 已开放 2 亿+ 预测结构，用 UniProt ID 直接查
uid = "P00533"  # 人类 EGFR 受体（癌症靶点）
meta = requests.get(f"https://alphafold.ebi.ac.uk/api/prediction/{uid}").json()[0]

pdb = requests.get(meta["pdbUrl"]).text  # 下载 3D 结构文件
# PDB 每行一个原子：ATOM 序号 原子名 残基 链 xyz坐标 ... B因子(存pLDDT)
atoms = [l for l in pdb.splitlines() if l.startswith("ATOM")]
print("原子数:", len(atoms))
# B-factor 列即 pLDDT：>90 高可信，<50 大概率是柔性无序区
plddt = [float(l[60:66]) for l in atoms]
print("平均置信度 pLDDT:", sum(plddt) / len(plddt))
常见误区 + 实践场景

误区：把 AlphaFold 当「实验替代品」。它给的是单一静态构象的高质量假设，不是真相——对无序蛋白、构象变化、突变效应、蛋白复合体的动态往往力不从心，pLDDT 低的区域尤其要当心。它加速但不取代实验验证。

BigCat 场景：这是「AI 把不可行变可行」的范式样本。你做跨学科思考时，可把它当元问题的锚：一个卡了半世纪的领域，被「序列→结构」这个映射一朝改写。类比到你自己的领域——哪些「难测但有海量间接信号（如进化冗余）」的问题，也在等一个 AlphaFold 时刻？

一句话 takeaway + 思考题

AlphaFold 的核心不是「更大的网络」，而是把进化冗余（MSA）当免费的约束信号——自然界早把答案写在物种间的共变里。💭 你手头哪个难题，其实早有「一起变化的冗余副本」在偷偷泄露答案？

神经算子与科学代理模型Neural Operators

科学计算PDE 求解GraphCast
一句话类比

把一个跑得极慢的数值模拟器物化成一次前向传播——就像给昂贵的实时查询建了个物化视图（materialized view）：预计算一次，之后查询近乎免费。传统天气/流体模拟要在超算上一步步迭代偏微分方程（PDE），神经算子学的是「输入场 → 输出场」的整体映射，绕开逐步迭代，一把出结果。

它解决什么问题 + 工作机制

痛点：气候、流体、材料应力这些系统由 PDE 描述，数值求解精确但慢且贵——全球天气预报要在超算上算一小时。而且传统解法「一次只解一个实例」，换个初始条件就得从头再算。

关键洞察是学「算子」而非「函数」：函数把数映到数，算子把整个函数映到整个函数（整张温度场 → 6 小时后的整张温度场）。傅里叶神经算子 FNO（Li et al., 2020）的招数很优雅：把物理场做 FFT 变到频域，只在低频模态上乘可学习权重（低频=大尺度结构），再变回来。因为权重活在频域、与网格分辨率无关，它能零样本超分辨率——低分辨率训练、高分辨率推理。

GraphCast（DeepMind, 2023, Science）是这条路的旗舰：用图神经网络学全球天气演化，10 天预报在 1 分钟内算完，多数验证指标上超过顶级传统数值预报系统。这不是「更准一点」，是成本降了几个数量级，让集合预报第一次变得便宜。

诚实边界：代理模型是插值器不是外推器——分布内飞快又准，遇到训练里没见过的极端情形（破纪录风暴）可能失灵。产业界普遍是代理模型 + 传统求解器混合。

代码示例
import torch
# 傅里叶层核心：在频域做线性变换 = 一次算子作用于整个场
def spectral_conv(x, weights):
    # x: (batch, channels, grid) 物理场，如温度/气压网格
    x_ft = torch.fft.rfft(x)                 # 变到频域：每个频率一个复系数
    modes = weights.shape[-1]                 # 只保留前 k 个低频（大尺度结构）
    out_ft = torch.zeros_like(x_ft)
    out_ft[..., :modes] = x_ft[..., :modes] * weights  # 频域逐点相乘=全局卷积
    return torch.fft.irfft(out_ft, n=x.shape[-1])  # 变回物理空间
# 关键：weights 活在频域，与网格分辨率解耦 → 可零样本超分辨率
常见误区 + 实践场景

误区：以为神经算子「理解了物理」。它学的是数据里的统计规律，不是守恒定律。所以它会违反能量守恒、在长时间滚动预测里漂移。别把插值当理解。

BigCat 场景：这个「用一次前向替换昂贵计算」的模式高度可迁移——凡是「同一套慢逻辑要反复跑在不同输入上」，都值得问：能不能训个代理模型？你熟悉的缓存/物化视图思想在科学计算里换了个数学外衣，骨架一样：拿空间和一次性训练成本，换查询时的速度。

一句话 takeaway + 思考题

神经算子学的是函数空间之间的映射，所以一次训练泛化到一整族问题，而非一个实例。💭 你系统里哪段「慢逻辑」其实每次都在解同一族问题、只是换了参数，适合被代理模型「物化」掉？

材料发现GNoME · Materials Discovery

计算化学图神经网络主动学习
一句话类比

化学组合空间大到天文数字，逐个用第一性原理精算（DFT）像对整张表做全表精确校验——准但慢到不可行。GNoME 的思路是先用一个廉价的 GNN 预测器当布隆过滤器（Bloom filter）：快速筛掉几乎肯定不稳定的候选，只把「可能命中」的少数送去做昂贵的 DFT 精算。廉价预筛 + 昂贵精验，标准的漏斗式过滤。

它解决什么问题 + 工作机制

痛点：新材料（电池、超导、催化剂）藏在巨大的元素组合空间里。判断一个假想晶体稳不稳定需要量子力学计算 DFT，一个候选几小时 CPU，穷举无望。历史上人类靠化学直觉猜——覆盖面窄。

GNoME（DeepMind, 2023, Nature）把晶体建模成图（原子=节点，化学键=边），GNN 学着预测「形成能」和能量凸包上方距离（e_above_hull，≈0 才稳定）。真正的引擎是主动学习闭环：GNN 筛出候选 → DFT 精算验证 → 新结果回填训练集 → GNN 变强 → 下一轮更准。这个自强化循环识别出约 38 万个新稳定晶体结构，很多是人类化学直觉想不到的。

GNoME 主动学习闭环（廉价筛 → 昂贵验 → 回填变强）
海量候选晶体 → GNN 快速筛稳定性 → 少量有希望候选
↓ 送去精算
DFT 量子力学验证 → 新稳定材料 + 标注 ↺ 回填训练集，GNN 下一轮更准
代码示例
# GNoME 思路：GNN 快速预测稳定性筛掉大多数，再交给 DFT 精算（漏斗）
from pymatgen.core import Composition

candidates = ["Li3PO4", "Na2FeO3", "CaTiO3", "Xe2Fe"]  # 海量候选的一角

def gnn_stability(formula):
    comp = Composition(formula)  # 真实 GNoME 吃晶体图，这里用占位分示意流程
    # e_above_hull：能量凸包上方距离(eV/atom)，越接近 0 越稳定
    return {"formula": formula, "e_above_hull": 0.02 * len(comp)}

# 廉价预筛：只让 <0.03 eV/atom 的候选进入昂贵的 DFT 阶段
to_verify = [c["formula"] for f in candidates
             if (c := gnn_stability(f))["e_above_hull"] < 0.03]
print("送 DFT 精算的候选:", to_verify)  # 从海量收敛到少数
常见误区 + 实践场景

误区：把「预测稳定」等同于「造得出、有用」。GNoME 给的是热力学稳定性假设，不保证能被合成、不代表有目标性能——后续有独立研究质疑这批新材料的新颖性与可合成性。计算发现是漏斗的入口，不是终点。

BigCat 场景：这个「廉价预筛 + 昂贵验证 + 结果回填」的主动学习闭环，是任何大规模筛选型决策的通用骨架——投资初筛、简历海选、实验候选排序。核心：别让昂贵环节去啃全量，让快模型先把漏斗收窄。

一句话 takeaway + 思考题

GNoME 的威力不在单个 GNN，而在主动学习闭环：预测→验证→回填让模型和数据集一起螺旋上升。💭 你的工作流里，有没有一个「验证结果」明明可以回填、却被白白丢掉的环节？

算法与数学发现AlphaTensor · FunSearch

离散搜索强化学习LLM 进化
一句话类比

前三个是 AI 加速科学，这个是 AI 产出新知识——发现人类没找到的算法与数学对象。做法是把「发现」重铸成搜索问题：AlphaTensor 把「找更快的矩阵乘法」变成一盘 AlphaZero 式单人棋；FunSearch 把 LLM 当变异算子、评估器当适应度函数 + 防幻觉门做进化搜索。像遗传算法，但「变异」由会写代码的 LLM 智能产生。

它解决什么问题 + 工作机制

痛点：找新算法/数学构造历来靠天才的灵光一现，不可规模化、不可复现。能不能让机器系统地搜？难点在搜索空间是离散的天文数字，且大多数候选是错的。

AlphaTensor（DeepMind, 2022, Nature）：矩阵乘法等价于张量分解，分解用的乘法次数越少算法越快。它把这变成单人游戏——状态=还没分解掉的张量，动作=选一个秩 1 项去消，用 AlphaZero 的蒙特卡洛树搜索来玩，对 4×4 矩阵找到了 50 年来第一个超越 Strassen 经典算法的方案。

FunSearch（DeepMind, 2024, Nature，名字=在函数空间里搜索）：用预训练 LLM 不断生成候选程序（不是直接生成答案），配确定性评估器打分。妙在——LLM 会幻觉，但评估器跑代码验证，错候选自动得低分被淘汰。它由此在组合数学的 cap set 问题上找到新解，并给在线装箱问题找到更优启发式。

FunSearch 进化循环（LLM 变异 + 评估器守门）
当前最优程序 → LLM 生成变体 (创造力来源)
↓
确定性评估器 (跑代码打分，幻觉在此被过滤) → 更优则保留 ↺

关键设计：输出是可验证的代码/构造而非自然语言断言——正确性由外部评估器保证，不靠 LLM 自己说了算，这类系统才「信得过」。

常见误区 + 实践场景

误区：以为「LLM 自己发现了数学」。真相是——创造力（LLM 生成）和正确性（评估器验证）被解耦。没有那个确定性评估器，LLM 生成的一堆「看着对」的东西全是幻觉。功劳属于这套架构，不是裸模型。

BigCat 场景：这个「LLM 大胆生成 + 程序严格验证」的解耦范式，是用 AI 做任何创造性但要求正确工作的黄金模板——让模型放开产点子，用你信得过的确定性检查器（测试、类型检查、事实核对）当闸门。别让 LLM 既当运动员又当裁判。

一句话 takeaway + 思考题

AI 发现新知识的秘诀是解耦创造与验证：模型负责「大胆猜」，确定性评估器负责「严格验」——幻觉在验证关被挡住。💭 你用 AI 时，有没有把「生成」和「验证」交给了同一方，等于让它自己给自己打分？

深入思考Deep Questions

1. 这四个案例里，AlphaFold/GNoME/GraphCast 是「加速」科学（更快得到人类本可求的答案），FunSearch/AlphaTensor 是「发现」新知识（人类没找到的）。这个区别在本质上是什么？
区别在验证的可得性。「加速」类有明确的真值可对照（真实蛋白结构、天气、DFT 精算），AI 逼近一个已定义的映射，本质是昂贵计算的压缩/插值——快，但不产出定义域外的新东西。「发现」类的解空间离散、组合爆炸、且有廉价的确定性验证器（算法对不对、构造是否满足约束，一跑便知）。正是「验证便宜、搜索昂贵」这个不对称，让搜索型 AI 能产出人类未及之物：不需要理解为什么，只需在天文数字空间里找到一个能通过验证的点。关键判据：你的问题有没有便宜又可靠的验证器？有 → 能让 AI 去「发现」；没有 → 只能在已有标注上「加速」。这也解释了为何数学/算法/材料稳定性等「可验证」领域最先被攻破，而「什么是好的科学理论」这类没有廉价验证器的问题仍属于人类。
2. AlphaFold 用「进化冗余（MSA）」、GNoME 用「主动学习回填」、FunSearch 用「评估器守门」。把这三个机制抽象成通用的工程模式，各对应你熟悉的什么系统思想？
三者都是「用便宜的信号源，撬动昂贵的目标」，撬点不同。① MSA 是免费的冗余副本挖约束——对应从多副本的一致/分歧推断拓扑（谁总和谁一起变=谁和谁耦合），或从日志共现反推服务依赖图。② 主动学习回填是把验证结果喂回训练、让筛选器自进化——对应自适应缓存/自调优：每次 miss 的真实代价被记录用来改进预取，越用越准。③ 评估器守门是生成与验证解耦——对应编译器类型检查、CI 测试门、金融对账：让高吞吐的一方自由产出，用可信但便宜的关卡挡错。合起来就是「AI for X」系统的通用配方：免费的弱信号（①）+ 自我强化的闭环（②）+ 便宜可靠的验证器（③）。缺哪个，系统就分别退化为「没先验、样本效率极低」「一次性、不进化」「幻觉失控、不可信」。
3. 神经算子/代理模型「快但不理解物理」，会违反守恒定律、在长程预测里漂移。这和 LLM「流利但会幻觉」是同一类问题吗？对「该在多大程度上信任统计模型」有什么启发？
是同构的问题：两者都是用高维统计拟合替代了显式的底层规则，因而在训练分布内惊艳、在分布外/长程外推时暴露「没有真正的因果/守恒约束」这一根本缺陷。GraphCast 的天气漂移 = LLM 的事实幻觉，都是「插值器被当成了推理器」。启发有三：① 信任绑定分布——问「输入离训练分布多远」而非「模型多强」；离得远，统计模型的自信毫无意义（呼应 Day 48 校准）。② 硬约束显式注入——物理信息神经网络把守恒律写进损失函数，正如给 LLM 套 RAG/工具/验证器外置事实约束；纯统计模型不会自己长出约束。③ 混合而非替代——「代理模型探索 + 传统求解器兜底」，正如「LLM 生成 + 确定性检查」。深层共识：统计模型负责广度与速度，符号/物理规则负责正确性与外推，二者互补而非替代。
4. 「把发现重铸成搜索」需要一个便宜的验证器。科学里哪些问题天然有这种验证器、哪些没有？这是否意味着 AI for Science 会系统性地偏向某类问题、而冷落另一类？
有廉价验证器的：可计算、可形式化、可仿真的问题——算法正确性（跑一遍）、数学构造（查约束）、蛋白结构（有实验/物理打分）、材料稳定性（DFT）。没有的：需昂贵长周期真实实验才能证伪的问题——新药的长期疗效、经济政策的社会后果、生态干预的十年影响、「什么是更好的科学范式」。这会造成系统性偏斜：进展集中在「验证便宜」的象限，让这些领域加速拉开，而「验证昂贵/无法形式化」的领域相对停滞——不因它们不重要，而因搜索型 AI 在那里没有可用的适应度函数。两个风险：① 度量扭曲议程——可被 AI 加速的问题吸走注意力和资金，「路灯下找钥匙」；② 验证器偏差被放大——DFT 或打分函数若有系统误差，AI 会高效过拟合到它的漏洞（呼应 Day 47 reward hacking）。元启示：判断「某领域会不会被 AI 快速改写」，先问的不是「模型够不够强」，而是「有没有便宜可靠的验证器」——这比模型能力更能预测变革次序。
5. 若「创造由 AI、验证由确定性系统」成为科学发现的主流范式，人类科学家的不可替代角色是什么？这对你追求「AI 超级个体」意味着什么样的能力重心转移？
搜索与验证被自动化后，人类价值向搜索循环之外三处上移：① 提出正确问题、定义正确验证器——AI 只在你给定的空间和适应度函数里最优化；选择优化什么是最高杠杆的人类判断，选错验证器，AI 会高效把你带向错误目标。② 跨域连接与品味——把「进化冗余」从生物搬到你的领域、看出远领域的同构，这种类比与审美是当前 AI 的弱项，也是你佛学×分布式×复杂性科学的组合优势。③ 为无法形式化的问题负责——价值权衡、伦理、「该不该做」，没有廉价验证器，只能由人承担。对「AI 超级个体」的能力重心转移：从「亲手求解」转向「设计求解系统」——核心竞争力不再是算得多快，而是会不会把模糊难题拆成「弱信号 + 自强化闭环 + 可信验证器」这套配方，并守住那个只有人能定的目标函数。AI 越能自动化「搜索」，人类「定义搜索什么、凭什么算数」的能力就越稀缺——这正是超级个体该押注的方向。
